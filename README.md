# Ultra - AI Notepad

A full-stack application for managing notes and tasks with AI integration.

This is a personal project to develop full-stack development skills, improve backend architecture, and explore AI integration with note-taking features.

**Frontend** created with [Lovable](https://lovable.dev)  
**Backend** built with FastAPI

## Project Structure

- **`frontend/`** - React + TypeScript application with Vite
- **`backend/`** - Python API (coming soon)

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The frontend will run at `http://localhost:5173`

### Backend

To be set up.

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite, Shadcn UI, TanStack React Query
- **Backend**: Python with FastAPI

## Development

- Frontend tests: `npm run test` (in `frontend/`)
- Linting: `npm run lint` (in `frontend/`)
